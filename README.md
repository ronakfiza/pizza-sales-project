# Pizza Sales Analysis Dashboard

## Project Overview

This Power BI project analyzes pizza sales data to uncover business insights, sales trends, customer preferences, and key performance indicators (KPIs). The dashboard helps stakeholders understand sales performance and make data-driven decisions.

## Objectives

* Analyze total revenue generated from pizza sales.
* Identify best-selling and least-selling pizzas.
* Track order trends over time.
* Understand customer purchasing behavior.
* Visualize key business metrics through interactive dashboards.

## Key KPIs

* Total Revenue
* Total Orders
* Average Order Value
* Total Pizzas Sold
* Average Pizzas per Order

## Dashboard Insights

* Top-selling pizza categories
* Revenue by pizza size
* Daily and monthly sales trends
* Best and worst performing pizza products
* Order distribution analysis

## Tools Used

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Files Included

* `PIZZA_SALES_PROJECT.pbix` – Power BI Dashboard File
* `README.md` – Project Documentation

## How to Open

1. Download the `.pbix` file.
2. Open it using Microsoft Power BI Desktop.
3. Explore the interactive dashboard and reports.
